# Style Sample Generation

This directory contains two Python scripts for generating SVG preview samples of handwriting styles:

## Scripts

### 1. `generate_style_samples_simple.py` (Currently Used)
- **Purpose**: Generates simple placeholder SVG previews for each style
- **Requirements**: None (pure Python, no dependencies)
- **Output**: Basic SVG files with text labels showing "Style X Preview"
- **Usage**:
  ```bash
  python3 generate_style_samples_simple.py
  ```
- **Location**: Saves to `model/style/style-{1-12}.svg`

### 2. `generate_style_samples.py` (For Real Handwriting)
- **Purpose**: Generates actual handwriting SVG previews using the AI model
- **Requirements**: All project dependencies (TensorFlow, etc.)
- **Output**: Real handwriting samples showing actual style characteristics
- **Usage** (when dependencies are installed):
  ```bash
  pip3 install -r requirements.txt
  python3 generate_style_samples.py
  ```
- **Note**: This script uses the full handwriting synthesis model to generate realistic previews

## Current Status

The UI currently uses placeholder SVG samples generated by `generate_style_samples_simple.py`. These are sufficient for the dropdown preview functionality.

To upgrade to real handwriting previews:
1. Install project dependencies: `pip3 install -r requirements.txt`
2. Run: `python3 generate_style_samples.py`
3. The SVG files in `model/style/` will be replaced with actual handwriting samples

## UI Integration

The style dropdown in the web UI (`/webapp/templates/index.html`) displays these SVG previews via:
- **Route**: `/api/style-preview/<style_id>` (in `webapp/routes/style_routes.py`)
- **CSS**: `.style-dropdown-custom` classes (in `webapp/static/css/main.css`)
- **JavaScript**: `loadStyles()` function (in `webapp/static/js/main.js`)

When you hover over or select a style in the dropdown, you'll see the SVG preview alongside the style name.
